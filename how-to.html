How to Use

// terminal
yarn add jazzer

// script.js
import jazzer from 'jazzer';

jazzer();

// style.css
#jazzer {
  transition: transform .5s, opacity .5s;
}

.jazzer-changing {
  opacity: 0;
  transform: scale(.9);
  pointer-events: none;
}

// index.html
<a href="some-page.html" data-jazzer-trigger>
  I trigger jazzer
</a>

<div id="jazzer">
  My contents will super-smoothly change
</div> You can even customize jazzer!
