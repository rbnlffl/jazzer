jazzer

jazzer lets you easily freshen up your page and improve its memorability by a bunch. You know these short white flashes when changing between pages? Ugggh, right? jazzer fixes that. It lets you control exactly what should visually happen when clicking on an internal link. Install it, import it, configure it. Give your project the sexyness it deserves!

Experience the jazzy smooth goodness!

How to Use

// cmd / bash
yarn add jazzer --save

// script.js
import jazzer from 'jazzer';

jazzer();

// style.css
#jazzer {
  transition: transform .5s, opacity .5s;
}

.jazzer-changing {
  transform: scale(.75);
  opacity: 0;
  pointer-events: none;
}

// index.html
<a href="some-page.html" data-jazzer-trigger>
  I trigger jazzer
</a>

<div id="jazzer">
  My contents will super-smoothly change
</div>

Advanced Stuff

If you want, you can also customize almost everything about jazzer! The example above illustrates how to use it with its defaults and produces the same transition you see on this site. When calling it, simply pass in an object containing your very own config.
